ButtonSwap3DS Builder is becoming 3DS Remap Builder! Because of the awesome work by Kartik on rehid, I am encouraging people to move over to using rehid instead of ButtonSwap3DS. His implementation is more full-featured than ButtonSwap3DS could ever be in its current state, and, to be honest, rehid is the direction that I would have had to go to get the features I wanted to implement.

As of this update of the builder (end of April 2021), the only reason I can recommend ButtonSwap3DS is that rehid does not yet have a gyroscope driver (meaning it won't work in games that use it). Beyond that, rehid has the same remapping capabilities as ButtonSwap3DS and supports remapping the controls handled by the IR module (C-Stick/nub and ZL/ZR).

For more information on rehid and getting it set up, you can see the GitHub project, the GBATemp thread, or u/MarioKartFan_123's excellent step-by-step guide.
